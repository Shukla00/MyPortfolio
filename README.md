## 🚀 How to Run Locally

If you'd like to check out the code and run it on your own machine:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Shukla00/MyPortfolio.git
    ```
2.  **Navigate to the directory:**
    ```bash
    cd MyPortfolio
    ```
3.  **Open in Browser:**
    Simply open the `index.html` file in any modern web browser.
    *Alternatively, use an extension like VS Code's "Live Server" for a better development experience.*
